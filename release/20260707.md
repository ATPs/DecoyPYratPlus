# DecoyPYratPlus v2.0.2

## Added

- Added a new `DecoyPYratPlus.digestion` module that centralizes the shared digestion implementation and can be run directly as a CLI.
- Added standalone digestion CLI support for FASTA input or `--sequence`, with `digest` and `trypsin` modes and `tsv`, `peptide`, and `fasta` output formats.
- Added focused `unittest` coverage for digestion behavior, CLI output, import wiring, and legacy-script reuse.

## Changed

- Moved the main digestion, trypsin, FASTA reading, fast-digest selection, and peptide-similarity helpers out of `utils.py` into the new `digestion.py` module.
- Updated `DecoyPYratPlus/decoyPYratPlus.py` to import digestion helpers from the new module while preserving the existing decoy-generation pipeline.
- Updated the legacy `DecoyPYratPlus/decoyPYrat.py` script to reuse the shared digestion implementation instead of keeping a separate local copy.
- Documented the standalone digestion CLI in `README.md`.

## Packaging

- Bumped package metadata in `setup.py` and `meta.yaml` to version `2.0.2`.
