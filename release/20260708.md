# DecoyPYratPlus v2.0.3

## Added

- Added Comet-style enzyme presets to the standalone `DecoyPYratPlus.digestion` CLI, including `Trypsin`, `Trypsin/P`, `Lys_N`, `No_cut`, and `Cut_everywhere`.
- Added richer digestion CLI help with concrete command examples, output samples, header-template examples, and the supplied Comet enzyme table.
- Added test coverage for enzyme presets, help text, header-template rendering, standalone I/L behavior, and main CLI compatibility.

## Changed

- Updated the standalone `digestion.py` CLI so `I` and `L` remain distinct by default; `--isobaric` now explicitly enables `I -> L` normalization.
- Kept the main `DecoyPYratPlus` and legacy decoy-generation CLIs on their previous default behavior, where digestion still normalizes `I -> L` unless `--no_isobaric` is set.
- Clarified the README to distinguish standalone digestion CLI behavior from the main decoy-generation pipeline.

## Notes

- Enzyme preset definitions in this release follow the user-supplied `COMET_ENZYME_INFO` table, including the `0`/`1` mapping to n-terminal and c-terminal cleavage behavior.
